# Food Delivery Microservices Application

A microservices-based food delivery system built using Spring Boot, Eureka Service Discovery, Docker Compose, and an Angular frontend.

## Tech Stack
- Java, Spring Boot
- Eureka Service Discovery
- MySQL, MongoDB
- Docker & Docker Compose
- Angular

## Services
- Eureka Server
- Restaurant Listing Service
- Food Catalogue Service
- Order Service
- User Service
- Angular Frontend

## How to Run
1. Start Docker
2. Run `docker-compose up`
3. Access:
   - Eureka Dashboard: http://localhost:8761
   - Frontend: http://localhost:4200
